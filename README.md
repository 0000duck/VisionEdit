# VisionEdit
基于Halcon的一款通用图像处理工具
###
1. 仿照VisionPro拖拉形式进行工具间值的传递及JOB流程运行走向
2. 工具之间采用插件，可动态加载调用
3. 目前图像处理算法使用Halcon，预期将Opencv也做成插件式工具进行调用
